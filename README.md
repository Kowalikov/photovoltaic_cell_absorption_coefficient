Praca inżynierska, kroki:


* Wyznaczenie materiałów do użycia na podłożu GaSb
* Wyznaczyć położenie wierzchołków pasm GaSb
* Wyrysować położenia wierzchołków pasm materiałów na podłożu GaSb
